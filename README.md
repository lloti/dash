### Contributing
1. Fork.
2. Create a feature branch (i.e. `git checkout -b <feature>`)
3. Commit.
4. Submit PR.

### Setup
1. Clone the repository
2. Run `npm install`
3. Rename `.env.sample` to `.env`
4. Run `gulp build` and finally `npm run start`
